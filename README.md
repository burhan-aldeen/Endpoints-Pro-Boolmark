# Endpoints-Pro-Bookmarklet 🚀
--- FELL FREE TO IMPROVE IT ----

A powerful JavaScript bookmarklet tailored for bug hunters and security researchers. It helps in extracting endpoints, URLs, and potential hidden paths from JavaScript files and HTML content while browsing.

## ✨ Features
* Extract all unique URLs and endpoints from the current page.
* Clean and organized output (Console or UI).
* Supports filtering for specific extensions (optional).
* Fast and runs directly in the browser.
